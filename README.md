# Flight Booking System

A full-stack flight booking system built using:

- Node.js
- Express
- MongoDB
- React
- JWT Authentication

## Features

- User registration
- Email verification
- Login with JWT authentication
- Flight search
- Flight booking
- Booking history

## Backend

Express REST API with MongoDB.

## Frontend

React application with pages:

- Register
- Login
- Verify Email
- Flights
- Bookings

## How to run

### Backend

cd backend  
npm install  
npm run dev  

### Frontend

cd test-app  
npm install  
npm start
